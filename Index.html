Contenido del Curso

Un viaje completo desde los fundamentos hasta tu primer proyecto.

Módulo 1: Fundamentos

Introducción a la programación, lógica, algoritmos, variables y herramientas esenciales.

Módulo 2: Control de Flujo

Domina las estructuras de decisión (if/else) y los bucles (for/while).

Módulo 3: Colecciones

Aprende a manejar listas, tuplas y diccionarios para organizar datos.

Módulo 4: Funciones

Organiza tu código en bloques reutilizables. Aprende sobre parámetros y librerías.

Módulo 5: POO

Introducción a la Programación Orientada a Objetos: clases, objetos y herencia.

Módulo 6: Archivos

Lee y escribe archivos de texto. Aprende a manejar excepciones y a depurar tu código.

Módulo 7: Tu Primer Proyecto Real

Es hora de aplicar todo lo aprendido. En el último módulo, te guiaremos para que construyas una aplicación completa desde cero.

  • Calculadora o Juego: Crea una aplicación de consola interactiva.
  • Gestor de Tareas: Diseña una pequeña aplicación para organizar tus pendientes.
  • Script de Automatización: Escribe código para automatizar una tarea sencilla. 
# Ejemplo: Gestor de Tareas en Python

tasks = []

def add_task(task):
    tasks.append({"name": task, "done": False})
    print(f"¡Tarea '{task}' añadida!")

def show_tasks():
    for i, task in enumerate(tasks):
        status = "✓" if task["done"] else " "
        print(f"{i+1}. [{status}] {task['name']}")

add_task("Aprender Python")
add_task("Construir un proyecto")
show_tasks()
